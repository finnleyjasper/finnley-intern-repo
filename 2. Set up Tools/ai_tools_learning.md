# Reflection
### Which tool you used
I have tried out Windsurf as my AI tool.

### What you used it for
I used it to explain and refactor a small script from a Unity project.

### What it helped with
It was very effective in explaining the code and its purpose. The refactoring was also quite good - though due to my current development environment, I was not able to test the refactored code. However, it was quite good at explaining why it made the changes it did, and I even learnt a more efficient way to write one of the functions my example code had included.

### What it struggled with
Sometimes it would repeat things that were not relevant to the question I had asked. For example, when I asked it to explain the refactored code I had it generate, it also explained every other function in the script. It seems that I need to be very clear and specific in my instructions.
